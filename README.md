# Smart Expense Tracker

A smart expense tracking system that uses machine learning to analyze spending patterns and provide insights.

## Features

- Add daily expenses with amount, category, date, and description
- Category-wise expense analysis
- AI-based pattern detection using clustering
- Monthly expense prediction using linear regression
- Smart savings suggestions
- Visual graphs and charts

## Technologies

- Python
- Streamlit for UI
- Scikit-learn for machine learning
- Pandas and NumPy for data processing
- SQLite for data storage
- Matplotlib and Plotly for visualizations

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Usage

Run the Streamlit app and use the sidebar to navigate between features.
